Summary

Proposes Random Walk Positional Encoding (RWPE) using k-step random walk landing probabilities as node features. Avoids LapPE's sign ambiguity but provides stochastic (non-deterministic) features. STORM's reachability profiles offer a deterministic alternative with native directed graph support.
