\section{실험 환경}
Apple M3 Pro 12코어, 18GB, Python 3.13.5, NumPy 2.4.3, SciPy 1.17.1, Cython 3.2.4.
STORM은 Cython C 확장(\texttt{-O3 -march=native})을 포함한 단일 구현체.
모든 수치는 3회 실행 최소값이며, 정확성은 원소별 비교로 100\% 확인.
