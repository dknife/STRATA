Summary

Introduces incremental arbitrary-order reachability computation using boolean matrices and path pruning. Uniquely supports directed and disconnected graphs. Achieves up to 11x speedup over NetworkX on synthetic and real-world networks. STORM extends this framework with sparse matrices, Cython acceleration, and dynamic support.
