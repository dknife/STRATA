1. Installation

Clone the Repository

git clone https://github.com/dknife/STORM.git
cd STORM/02_STORM_Implement

Install Dependencies

pip install -r requirements.txt

Required packages: numpy, scipy, networkx, tqdm

Install Cython (for C acceleration)

pip install cython setuptools

2. Compile Cython Extension

The Cython C-extension provides 1.7–5.5× additional speedup by fusing pruning operations into a single C loop.

python setup.py build_ext --inplace

This compiles storm/_storm_core.pyx → storm/_storm_core.cpython-*.so with -O3 -march=native.

If Cython is not available, STORM falls back to pure Python automatically.

3. Usage

Basic APSP

# Sparse STORM APSP on a real-world edge-list
python main.py -m storm -i path/to/graph.txt -r

# With hop constraint (k=5)
python main.py -m storm -i path/to/graph.txt -r -k 5

# Compare all methods
python main.py -m compare -i path/to/graph.txt -r

Python API

from storm.core_cython import cython_storm_apsp
from storm.loader import load_graph

# Load graph
A, n, m = load_graph('graph.txt', fmt='edgelist', directed=False)

# Full APSP
D = cython_storm_apsp(A, verbose=True)

# Hop-constrained (k=5)
D_approx = cython_storm_apsp(A, k=5, verbose=True)

# Dynamic edge addition
from storm.dynamic import DynamicStorm
ds = DynamicStorm(A, D)
ds.add_edge(u, v)  # 22-33x faster than recomputation

4. Running Experiments

Main Benchmark

cd ../03_experiment_result
python run_benchmark.py

This runs all six experiments (Facebook, BA scalability, topologies, hop-constrained, progressive sparsity, D-STORM) and saves results to storm_results.json.

Grid Visualization

cd ../03_a_experimental_result
python run_grid_visual.py

Generates distance matrix heatmaps comparing STORM (k=2,4,8,16,32) with SciPy exact APSP on a 20×20 grid.
