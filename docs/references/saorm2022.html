Summary

Extends AORM with sparse matrix operations (S-AORM). A 4-page conference paper that demonstrates speedup over the original dense implementation but lacks multi-baseline comparison and topology analysis. STORM builds upon this work with Cython acceleration, correctness fixes, and comprehensive evaluation.
